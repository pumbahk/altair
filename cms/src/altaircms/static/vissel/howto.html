各購入方法・引取方法　詳細

1.ヴィッセルチケットにてクレジットカードでお支払い

  • 予約受付完了と同時に決済され、即時予約確定となります。 支払回数は≪1回払い≫のみのお取扱いとなります。
  • 右記のクレジットカードがご利用できます。


2.QRコードを印刷/メール送信

  • 購入完了画面または、購入履歴確認画面より
    • 「QR表示（印刷）」…「印刷」ボタンをクリックすると、QRコード表示されたページが開きますので、印刷し試合当日にお持ちください。
    • 「メール送信」…「メール送信」ボタンをクリック、ご指定のメールアドレス(パソコン・携帯いずれも可)へQRメール送信。QRコードが表示された画面をメモ/保存をして試合当日ご来場ください。
      • クレジットカードの携帯へQRコードを送信される場合は、@ticketstar.jp からのメールが受信できるよう、ドメイン指定の設定をお願いいたします。
      • QRコードは１席につき１つずつ発行されます。会場内にてお待ち合わせをされる場合などには、相手の方に事前にQRコードをお渡し頂きますようお願いいたします。QRコードは、上記のメール送信の手順にて送信後、受信したメールを相手の方に転送することでお渡しいただけます。

3.試合当日、QRコードをお持ち頂き、会場にてチケットと引換

  • 試合当日、チケット窓口にてQRコード（印刷したもの、或いは携帯画面）をお見せください。
  • 係員が、ご購入のチケットとお引換、お渡しいたします。

2.セブン-イレブンにて引取

  • 購入完了メール内の「引換票番号」をお控え下さい。
  • 引換票番号は、「購入完了画面」および「購入履歴確認画面」より印刷も可能です。
  • 引取受付期間内に、引換票または引換票番号をお持ちの上、セブン-イレブン店頭レジにてチケットをお引取り下さい。
  • セブン-イレブンでの引換には、発券時に発券手数料が105円/1枚かかります。
  • 試合当日は、セブン-イレブンにて発券されたチケットにてご入場となります。

1.セブン-イレブンにてお支払い

  • 購入完了メールに記載された予約有効期日までにセブン-イレブン店頭にてお支払頂きます。 お支払が確認できた時点で予約確定となります。（通常は、予約日時から3日間が予約有効期間となります）
  • 有効期日までにお支払いいただけなかった場合は、予約は無効（キャンセル）となります。ご注意ください。
  • お支払いの際は、購入完了メール内の「払込票番号」をお控えの上、セブン-イレブン店頭レジにて代金をお支払いください。
  • 払込票番号は、「購入完了画面」および「購入履歴確認画面」より確認、印刷も可能です。
  • セブン-イレブン支払いでは、決済手数料（158円/1申込）がかかります。

2.セブン-イレブンにて引取

  • 購入完了メール内の「引換票番号」をお控え下さい。
  • 引換票番号は、「購入完了画面」および「購入履歴確認画面」より印刷も可能です。
  • 引取受付期間内に、引換票または引換票番号をお持ちの上、セブン-イレブン店頭レジにてチケットをお引取り下さい。
  • セブン-イレブンでの引換には、発券時に発券手数料が105円/1枚かかります。
  • 試合当日は、セブン-イレブンにて発券されたチケットにてご入場となります。
