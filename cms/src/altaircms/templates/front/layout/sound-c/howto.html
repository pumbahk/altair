お申込み方法

公演毎にお申込方法が異なりますの下記をよくお読み頂きお申込下さい。

  1. 公演一覧よりご希望の公演をクリック。
  2. ご希望の枚数・決済方法を入力し、「申込｣をクリック。
  3. お申込の内容を確認し、「次へ」をクリック。
  4. お客様情報等、必要項目を入力し、「申込」をクリック。
  5. 「確認する（ここをクリックしてください）」をクリック。
  6. 申込完了！！

抽選エントリー時お申込方法

  1. 公演一覧よりご希望の公演をクリック。
  2. ご希望の枚数・決済方法を入力し、「申込｣をクリック。
  3. お申込の内容を確認し、「次へ」をクリック。
  4. お客様情報等、必要項目を入力し、「申込」をクリック。
  5. 「確認する（ここをクリックしてください）」をクリック。
  6. 申込完了！！　※「抽選の申込を受付ました。」というメッセージが出たことを確認してください。

注意事項

お引換の際に予約番号とお客様の登録された電話番号が必要になります。忘れないようにメモなどをご用意ください。
チケットは画面に掲出されたチケット引換期間内に引き換えられないとキャンセルになりますのでご注意ください。

  • お客様登録情報に関するお問い合わせは一切お受けできません。
  • 公演毎に枚数制限を設定させて頂く場合がございますのでご了承下さい。
  • ドメイン指定受信、メール指定受信ご利用の方は、｢lawsonticket.jp｣、｢l-tike.com｣、｢lhe.lawson.co.jp｣を受信できるよう必ず設定してください。
  • 予定枚数になり次第、終了になります。

抽選エントリー時お申込方法

当選・落選はご登録のメールアドレスに通知いたします。
当選発表日翌日になってもメールが来ない場合は必ずメール当落発表期間内に、左メニュー内「チケット申込確認」からもご確認ください。
また、当選落選は本サイト内でのみの確認となり、ローソンチケットドットコムでは確認できませんのでご注意ください。

  • お客様登録情報に関するお問い合わせは一切お受けできません。
  • 当選確認のための電話番号・パスワードのお問い合わせは一切お受けできません。
  • 当選後のキャンセル、変更はできません。
