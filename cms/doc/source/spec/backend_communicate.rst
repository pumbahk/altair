バックエンドとの通信方式
======================================

呼び出し方式
---------------------

バックエンドはイベント情報をCMSに通知する。オペレーションのタイミングを通知トリガとする。

シリアライザ
---------------------

JSONを使用する。

e.g.:

.. code-block:: python

   {
       'object_name': {
           'integer': int,
           'string': unicode,
           'list': [a, b, c]
       }
   }

エラー処理
------------------

基本方針として5xxならリトライ、4xxならリトライせずメール送信。
