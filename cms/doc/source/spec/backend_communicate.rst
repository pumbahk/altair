バックエンドとの通信方式
======================================

呼び出し方式
---------------------

CMSは、バックエンド側に設けたRESTful APIを用いて情報の取得・更新を行う。
CMS側の情報操作が必要な場合、同様にRESTful APIを実装する。


シリアライザ
---------------------

JSONを使用する。

e.g.:

.. code-block:: python

   {
       'object_name': {
           'integer': int,
           'string': unicode,
           'list': [a, b, c]
       }
   }

エラー処理
------------------

API個別に検討する。
基本方針として5xxならリトライ、4xxならリトライせずメール送信。
