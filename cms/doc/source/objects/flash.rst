.. _object-flash:

Flashオブジェクト
--------------------------

Flashを表示するのに必要なオブジェクト。アセットのメタデータをそのままマッピングする。

.. code-block:: javascript

 {
     'flash': {
         'url': '', // Flashファイルパス
         'title': str, // タイトル
         'mimetype': str // content-type
     }
 }


see also: :ref:`widget-flash`
