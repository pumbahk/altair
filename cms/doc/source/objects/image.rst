.. _object-image:

画像オブジェクト
---------------------------

画像を表示するのに必要なオブジェクト。アセットのメタデータをそのままマッピングする。

.. code-block:: python

 {
     'image': {
         'url': str,
         'alt': unicode,
         'size': int,
         'width': int,
         'height': int,
     }
 }


see also: :ref:`widget-image`
