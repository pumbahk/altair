altair README
-------------

Setup
=====
::

  $ mkdir altair-devel
  $ cd altair-devel
  $ git clone git@github.com:ticketstar/altair
  $ easy_install virtualenv
  $ virtualenv env
  $ env/bin/easy_install pyramid
  $ cd vega
  $ ../env/bin/python setup.py develop
  $ ../env/bin/paster serve development.ini --reload
